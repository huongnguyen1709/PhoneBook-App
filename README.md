<!-- GETTING STARTED -->

### Installation

1. Clone the repo

```sh
git clone https://github.com/huongnguyen1709/PhoneBook-App
```

1. Install NPM packages

```sh
npm install
```

3. Implement application on local server

```sh
npm start
```
